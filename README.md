# Tangelo

## To Run

```
# Requires https://github.com/tomekwi/elm-live
# Runs on http://localhost:8000

./bin/run

# which does:

elm-live main.elm --output=public/js/main.js --dir=public/
```

## Just To Make

```
elm-make main.elm --output=public/js/main.js

# then upload /public to wherever
```
